# React 중급과정 일기장 프로젝트
## 개발순서
### 개발 순서
1. MainView와 HistoryView의 화면 전환 구현
2. API에서 질문 리스트 가져오기 + 날짜 표시
3. 일기 작성 기능 + 자동 저장 기능 구현 (복수 날짜 정보를 저장할 수 있는 형태로 구현)
4. 히스토리 화면 구현 (답변 상태 위치를 App 컴포넌트로 올리기)
